Potato Rage
-----------

A game about potatoes that I made last year.

It's a JS game made from scratch (without framework) that heavily uses touch gestures.

The graphics aren't very polished and it's a bit laggy on low-end devices.


Live Version
------------
You can test the game [here](http://venam.nixers.net/potato)
